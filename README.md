# Toolbox 小工具集

一個簡約大地色系的小工具 Dashboard，集中放置日常常用工具。

## 工具列表

- 🔗 縮短網址 — 接入 TinyURL API
- 🍅 番茄時鐘 — Pomodoro Technique 計時器
- ⊞ QR Code 產生器 — 即時生成可下載

## 技術

- 純 HTML / CSS / JS，單檔案
- PWA 支援（可加到手機主畫面）
- 無框架依賴，無後端

## 部署

本項目使用 GitHub Pages 部署，推送到 `main` branch 自動發布。

## 日後加工具

在 `index.html` 的 `tools-grid` 新增工具卡片，並新增對應 `page-xxx` 即可。
